Orion Edge Systems

Platform Engineering Intern, Device Cloud

San Mateo, CA Fall 2026 Internship Hybrid $38/hr - $48/hr

Orion Edge Systems builds secure cloud software for connected field devices used by operations teams around the world. As a Platform Engineering Intern, you will help develop backend services, data pipelines, and APIs that keep distributed device fleets observable, reliable, and easy to configure.

What You Will Do

  • Build scalable distributed systems that process high traffic from hundreds of thousands of devices.
  • Improve high-concurrency key-value workflows using Redis, DynamoDB, caching, and idempotent job queues.
  • Design Postgres schemas, optimize SQL queries, and support relational data models for device operations.
  • Use Kafka, SQS, and structured logging to move events through reliable backend pipelines.
  • Deploy services with Terraform, Docker, Kubernetes, and GitHub Actions.
  • Create low-latency, high-throughput, high-reliability microservice architectures.
  • Develop remote configuration tools that safely update products across multiple regions.
  • Prototype computer vision integrations that turn device signals into actionable alerts.
  • Design clean REST APIs and implement production features in Python, Go, or TypeScript.

What You Bring

  • Currently pursuing a Bachelor's or Master's degree in Computer Science or a related technical field.
  • Expected graduation between December 2026 and June 2027.
  • Previous internship, research, or project experience shipping backend or cloud features.
  • Familiarity with Python, Golang, Linux, operating systems, AWS, and distributed systems concepts.
  • Comfort working in an agile software development environment with code reviews and clear communication.
  • Experience with REST APIs, observability, testing, and production debugging is a plus.
